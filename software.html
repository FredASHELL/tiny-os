Third-Party Software

No external software has been developed for TinyOS.

If you want to create software for TinyOS, you are welcome to try. However, you will probably need to make significant changes to the code.

What Might Be Needed?

A program would likely require files such as:

  • boot.asm
  • kernel.asm

The exact requirements may change as TinyOS develops.

Should You Make Software?

Honestly, it is probably not worth it right now.

Future versions of TinyOS will likely contain more built-in commands and features to experiment with.

Developer Advice

Yes.

Maybe.
