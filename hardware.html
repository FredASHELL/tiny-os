Hardware Support

So far, TinyOS has not been tested on real hardware. All testing has been done using VirtualBox 6.1.

If you want to run TinyOS on a real computer, 64 MB of RAM should be more than enough. It can probably run with less, but nobody has checked.

TinyOS is a 16-bit operating system, so you will need a machine capable of running 16-bit software.

Boot Media

  • Floppy Disk - Recommended
  • USB Floppy Drive - Probably works
  • USB Drive - Unknown
  • CD/DVD - Unknown

When developing TinyOS, the operating system is usually booted from a floppy disk image (.img) inside VirtualBox.

Real Hardware Testing

If you care that much, put it on an old computer and see what happens.
